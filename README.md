# 基金日报 · Fund Daily

基于 AI 技能自动生成的每日基金日报，覆盖 A 股行情、全球市场、资金流向与板块概率预估。

- **在线地址**：[fund-daily](https://luojinbin.github.io/fund-daily/)
- **技能路径**：`skills/market/fund-daily/SKILL.md`
- **定时生成**：每个工作日 18:32 自动触发

## 日报板块

| 编号 | 板块 | 说明 |
|------|------|------|
| [01] | 全球市场概览 | A 股指数 + 美股 + 港股 + 商品期货 + 汇率 |
| [02] | 全球快讯 | 5 条精选快讯，含来源链接 |
| [03] | 基金行业热点 | 6 条行业热点，含标签与摘要 |
| [04] | 今日操作建议 | 5 条操作建议 |
| [05] | 板块资金流向 | 29 个概念板块，多源交叉验证，ECharts 图表 |
| [06] | 次日板块涨跌概率预估 | 多因子模型，表格 + 色阶 |
